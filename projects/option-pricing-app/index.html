Overview

This project is an interactive option pricing app built to explore how theoretical option values change as inputs like stock price, strike price, volatility, time to expiration, and the risk-free rate vary.

What It Does

  • Computes theoretical option prices using standard pricing logic
  • Lets users change key inputs and see how prices respond
  • Provides an applied way to build intuition for derivatives and volatility sensitivity
  • Turns abstract finance concepts into something visual and interactive

Why I Built It

I built this project to strengthen both my quantitative finance intuition and my ability to turn mathematical ideas into usable tools. For quant trading, I think that bridge between theory and implementation is really important.

Key Takeaways

  • Option prices are highly sensitive to volatility and time
  • Interactive tools are useful for developing intuition beyond formulas alone
  • Building even a simple pricing app requires careful handling of assumptions and inputs

Tech Stack

  • Python
  • Streamlit
  • Numerical / quantitative finance logic
